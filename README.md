# Pokemon-App
This is a simple Android app built with Kotlin that fetches Pokemon data from API and displays it.

## Features

- MVVM
- Retrofit
- Navigation component (Single activity)
- Room
- Glide
- Coroutines
- dagger-hilt
- CardView
